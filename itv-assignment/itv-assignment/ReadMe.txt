
Необходимо разработать программу на С++, выполняющую следующую последовательность действий:

1. Получение списка имен текстовых файлов в указанной папке;

2. Организация пула потоков (количество потоков в пуле задается константой в программе)
обработки полученного списка файлов;

3. Каждый поток извлекает одно имя из набора, подсчитывает количество слов в файле и
обновляется общую (по всем файлам) сумму слов. При извлечении имя файла удаляется из
набора.
